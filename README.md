# king3.me

> This is a highly personalized personal blog.

# ⚙️ Tech Stack

- **Framework**: React + Next.js
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Framer Motion
- **Database**: Supabase
- **CMS**: Payload
- **Deployment**: Vercel
